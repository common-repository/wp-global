Using JDS WP Global plugin, you can automatically publish your content in other languages of your choice. IT NEEDS AN ACTIVE GOOGLE TRANSLATE API SUBSCRIPTION.

== Description ==

You are missing on 46% global traffic by not publishing your content in languages other than English! This plugin changes it all!

This plugin allows you to automatically post your english content to other languages of choice whenever you publish a new post. Take your Wordpress blog to a new level to a whole new audience :)

However this needs an active Google Translate API Subscription which is worth because of the traffic you will gain by publishing your post in other languages. If you get any issues/bugs, feel free to head over http://www.2php.info/wordpress/wpglobal and leave a comment & it should be fixed as soon as possible.

And all you have to do is just install this plugin to get started! For more details, visit http://www.2php.info/wordpress/wpglobal


== Installation ==

Installing JDS WP Global Wordpress plugin is very easy. Just copy the "jdswpglobal" folder & paste it into your wp-content/plugins folder.
For detailed instruction on how to install manually, visit http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation

After installing the plugin, the next thing you need to do is activate the plugin by logging in to your Wordpress blog admin panel & clicking on installed plugins 
sub menu link. Once you activate JDS WP Global plugin, you will find a new tab. Just click on that tab & follow the instructions there to use the plugin.

For any help, visit us at http://www.2php.info/wordpress/wpglobal
